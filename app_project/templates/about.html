About BookLog

A complete bookstore recommendation demo.

BookLog combines a real catalog interface, account flow, rating storage, and a hybrid recommendation model.

Recommendation engine

Content-based filtering compares book metadata, collaborative filtering learns from reader behavior, and popularity fallback keeps results useful for new users.

Application features

The app includes registration, login, profile recommendations, catalog search, collection filters, and persistent ratings in SQLite.
