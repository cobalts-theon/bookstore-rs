# Final Project

Repository này đã được tách thành 2 phần rõ ràng:

```text
app_project/        # Source code chính của website BookLog
report_materials/   # Báo cáo, slide, tài liệu thuyết trình
```

## Chạy dự án chính

```powershell
cd app_project
python -m pip install -r requirements.txt
python app.py
```

Sau đó mở:

```text
http://127.0.0.1:5000
```

## Xem tài liệu báo cáo

Các file báo cáo và slide nằm trong:

```text
report_materials/docs/
```
